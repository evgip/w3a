# Модуль Content

Модуль для обработки контента: Markdown, @упоминания, эмодзи, хэштеги.

## Возможности

- ✅ Парсинг Markdown через Parsedown
- ✅ Защита от XSS (SafeMode)
- ✅ Обработка @упоминаний
- ✅ Автоматическое добавление target="_blank" для внешних ссылок
- ✅ Кэширование результатов
- ✅ Два режима: полный (посты) и ограниченный (комментарии)
- ✅ Фильтрация по чёрному списку доменов

## Использование

### Базовый парсинг

```php
use App\Modules\Content\Core\Markdown;

// Полный режим (для постов)
$html = Markdown::parse($text);

// Ограниченный режим (для комментариев, без картинок)
$html = Markdown::parseComment($text);

// Простой текст (без Markdown)
$html = Markdown::parsePlain($text);
